👋 ¡Bienvenido al Curso de Inteligencia Artificial!

Esta guía te va a acompañar paso a paso para que tengas todo listo antes de la primera clase.
No te apures: vamos despacio y de a poco.

📑 Lo que vamos a hacer juntos:

  1. Crear tu cuenta en GitHub (donde están las clases)
  2. Descargar el material del curso a tu computadora
  3. Preparar tu computadora para usar las clases
  4. Comprobar que todo funciona
Antes de empezar, asegurate de tener conexión a Internet estable y, si es posible, la computadora enchufada a la corriente. Vamos a descargar varios programas.

🟦 Parte 1: Crear tu cuenta en GitHub

GitHub es como una gran biblioteca en Internet donde guardamos los archivos del curso. Necesitás una cuenta gratis para poder descargarlos cómodamente.

1Abrí tu navegador (Chrome, Edge, Safari o Firefox) y andá a la dirección:

https://github.com/signup

2Vas a ver una pantalla que dice "Sign up to GitHub". Te va a pedir, en este orden:

  • Email: escribí tu correo electrónico (el que usás todos los días).
  • Password: inventá una contraseña con al menos 15 letras, o 8 con un número y un símbolo.
    👉 Anotala en un papel para no olvidarla.
  • Username: elegí un nombre de usuario (puede ser tu nombre + apellido, ej: maria-perez). Si está ocupado, GitHub te avisa.
  • Email preferences: podés escribir n (por "no") si no querés recibir publicidad.

3Vas a tener que resolver un pequeño juego visual (un captcha) para demostrar que no sos un robot. Seguí las instrucciones en pantalla.

4GitHub te va a enviar un código de 6 u 8 dígitos a tu correo. Abrí tu casilla de email, copiá el código y pegalo en la página.

Si no te llega en 1 o 2 minutos, revisá la carpeta "Correo no deseado" o "Spam".

5¡Listo! Ya tenés tu cuenta. Si te pregunta cosas como "¿cuántas personas trabajarán contigo?", podés elegir "Just me" (solo yo) y "Student" (estudiante). En la pantalla siguiente podés hacer clic en "Skip personalization" para saltar esos pasos.

Anotá tu usuario y contraseña en un cuaderno. Los vamos a usar varias veces.

🟩 Parte 2: Descargar el material del curso

Hay dos caminos para descargar el material. Elegí el más fácil para tu computadora:

📥 Opción A: Camino fácil para todos (Windows y Mac)

1Andá a la dirección del curso:

https://github.com/TataInti/clases

2Buscá un botón verde que dice <> Code (arriba a la derecha de la lista de archivos). Hacé clic ahí.

3Se abre un menú. Hacé clic en "Download ZIP" (la última opción).

4Se va a descargar un archivo llamado clases-main.zip en tu carpeta Descargas (o Downloads).

5Descomprimí el archivo:

  • En Windows: hacé clic derecho sobre el archivo → "Extraer todo..." → "Extraer".
  • En Mac: hacé doble clic sobre el archivo. Se descomprime solo.

Te recomiendo mover la carpeta resultante (clases-main) a un lugar fácil de encontrar, por ejemplo tu Escritorio o tu carpeta Documentos.

Esta opción es la más rápida, pero si algún día querés actualizar el material vas a tener que volver a descargar el ZIP. Si te animás a algo más profesional, mirá la Opción B.

📥 Opción B: Usando Git (recomendada para futuro)

Esta opción permite actualizar el material con un solo comando. Es un poquito más técnica.

1Instalá Git según tu sistema:

  • Windows: descargalo desde git-scm.com/download/win y aceptá las opciones por defecto.
  • Mac: abrí la Terminal (buscala en Spotlight con ⌘+Espacio) y escribí git --version. Si no lo tenés, te va a ofrecer instalarlo.

2Abrí una terminal:

  • Windows: botón Inicio → escribí PowerShell → Enter.
  • Mac: abrí la app Terminal.

3Escribí estos comandos, uno por uno (apretá Enter después de cada uno):

cd Documents
git clone https://github.com/TataInti/clases.git
cd clases

🟧 Parte 3: Preparar tu computadora

Ahora vamos a instalar las herramientas que necesita el curso. Elegí la pestaña según tu computadora:

🪟 Si usás Windows
🍎 Si usás Mac

🪟 Instrucciones para Windows

Hay un asistente automático que hace TODO por vos: instala Python, Visual Studio Code, descarga las clases y prepara las herramientas. Es la opción más recomendada.

1Hacé clic en el botón Inicio (esquina inferior izquierda).

2Escribí PowerShell. Cuando aparezca "Windows PowerShell", hacé clic derecho sobre él y elegí "Ejecutar como administrador".

Windows te va a preguntar si permitís cambios. Hacé clic en "Sí".

3Se abre una ventana azul. Copiá el siguiente texto completo y pegalo dentro (clic derecho del mouse pega), después apretá Enter:

powershell -NoProfile -ExecutionPolicy Bypass -Command "[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12; & ([scriptblock]::Create((Invoke-RestMethod 'https://raw.githubusercontent.com/TataInti/clases/main/scripts/setup_windows.ps1')))"

4Esperá pacientemente. El asistente puede tardar entre 10 y 30 minutos según tu conexión. Vas a ver muchos textos pasar; es normal.

5Cuando termine, se abrirá Visual Studio Code con las clases listas. ¡Felicitaciones, ya está todo!

Las clases quedan guardadas en:
C:\Usuarios\TU_NOMBRE\GitHub\clases

🍎 Instrucciones para Mac

En Mac vamos a hacer los pasos de a uno. Tranquilo, son rápidos.

1Abrí la Terminal: apretá ⌘ + Espacio, escribí Terminal y dale Enter.

2Verificá si tenés Python. Escribí:

python3 --version

Si te muestra algo como Python 3.12.x, ya está. Pasá al paso 4. Si no, seguí al paso 3.

3Instalá Homebrew y Python. Pegá estos comandos uno por uno:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Cuando termine (puede tardar varios minutos):

brew install python@3.12

4Entrá a la carpeta del curso (la que descargaste en la Parte 2):

cd ~/Documents/clases-main
Si descargaste con Git, la carpeta se llama clases en lugar de clases-main. Ajustá el comando.

5Creá un entorno aislado (es como una "caja" donde se instalan las herramientas sin afectar el resto de tu Mac):

python3 -m venv .venv
source .venv/bin/activate

Vas a notar que aparece (.venv) al inicio de cada línea. Eso quiere decir que el entorno está activado. ✅

6Instalá las herramientas de Inteligencia Artificial:

pip install -r requirements.txt

Esta parte puede tardar entre 10 y 20 minutos. No cierres la ventana hasta que vuelva a aparecer la línea con (.venv) al principio.

7Instalá Visual Studio Code (para ver las clases más cómodo) desde:

https://code.visualstudio.com/Download

Descargá la versión Mac Universal, abrí el archivo .zip y arrastrá Visual Studio Code a la carpeta Aplicaciones.

🟪 Parte 4: Comprobar que todo funciona

1Abrí Visual Studio Code.

2En el menú de arriba, andá a File → Open Folder (o Archivo → Abrir carpeta) y elegí la carpeta clases (o clases-main) que descargaste.

3Vas a ver una lista de carpetas (modulo_1, modulo_2, etc.). Entrá en modulo_1 y abrí el archivo clase_1_que_es_la_IA.ipynb con un doble clic.

4La primera vez te puede aparecer un mensaje preguntando si confiás en los autores. Hacé clic en "Yes, I trust the authors" (Sí, confío).

5Si arriba a la derecha te pide "Select Kernel", elegí .venv (Python 3.12).

Si ves el contenido de la clase con texto, imágenes y bloques de código, ¡todo está perfecto! Ya estás listo para la primera clase. 🎉
